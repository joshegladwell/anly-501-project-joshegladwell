Data Gathering

First and foremost, I am using a dataset from the Center for Homeland Defense and Security (CHDS) that tracks "shooting incidents at K-12 schools (Jan 1970 - Jun 2022)." I downloaded the data from this website as an excel file with a "README" in its first tab and four tables in the remaining tabs: incident, shooter, victim, and weapon. I exported these tables as csv files and created a README.txt, which can be found in my GitHub repo here.
