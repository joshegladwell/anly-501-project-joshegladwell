Data Gathering

K-12 School Shooting Database - Center for Homeland Defense and Security

First and foremost, I am using a dataset from the Center for Homeland Defense and Security (CHDS) that tracks "shooting incidents at K-12 schools (Jan 1970 - Jun 2022)." I downloaded the data from this website as an excel file with a "README" in its first tab and four tables in the remaining tabs: incident, shooter, victim, and weapon. I exported these tables as csv files and created a README.txt, which can be found in my GitHub repo here. The data are formatted as follows:

Mass Shooter Database - The Violence Project

I am also using a dataset from The Violence Project, a nonprofit research center that provides access to their full database on request. Subject to their terms of use, I am unable to publicize any data from the Mass Shooter Database, but access to this data can be requested at this link.

APIs

I am using APIs from Twitter, 4chan, Reddit, Google Trends, and OpenWeather to supplement my dataset.
